## Описание проекта

REST-API сервис управления иерархическими группами людей.

Группы образуют дерево (у группы могут быть дочерние группы), люди привязаны к
группам. Сервис умеет считать участников как в самой группе, так и рекурсивно
вместе с дочерними, а также выводить списки людей по тем же двум режимам.

### Технологический стек

- **Go 1.25**
- **PostgreSQL 16** (`pgx/v5`)
- **REST** + контракт **OpenAPI 3.0** (`oapi-codegen`), роутер `chi`
- Миграции — `golang-migrate` (с тестовыми данными)
- Docker + docker-compose
